安装

Node版本要求

Fvx 需要 Node.js v12.16.1 或更高版本 (推荐 v12 以上)。

可以使用下列任一命令安装这个新的包:

npm install -g fvx@latest# OR yarn global add fvx@latest

安装之后，你就可以在命令行中访问fvx命令。你可以通过简单运行fvx，看看是否展示出了一份所有可用命令的帮助信息，来验证它是否安装成功。

你还可以用这个命令来检查其版本是否正确：

fvx --version

升级

如需升级全局的Fvx包, 请运行:

npm update -g fvx@latest#ORyarn global upgrade --latest fvx
